## ‚úÖ Quick Reference

### Setup Commands

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```

### Test Commands

**Backend:**
```bash
cd backend
pytest
pytest --cov=.
```

**Frontend:**
```bash
cd frontend
npm test
npm test -- --coverage
```

### URLs
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
- Health Check: http://localhost:8000/api/health
